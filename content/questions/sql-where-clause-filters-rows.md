Which `SQL` clause is used to filter rows in a `SELECT` statement?

## Options

A. WHERE
B. SELECT
C. GROUP BY
D. ORDER BY

## Expected answer

A. WHERE

## Hints

- This clause is used before any grouping operations.
- It's typically placed after the FROM clause in a SELECT statement.
