Which of the following is a characteristic of a slowly changing dimension (`SCD`) Type 2 in data warehousing?

## Options

- [ ] A. Overwriting the old value with the new value
- [ ] B. Adding a new column for the new value
- [ ] C. Creating a new row for each change, preserving historical data
- [ ] D. Maintaining separate current and historical tables

## Expected answer

- [x] Creating a new row for each change, preserving historical data

## Hints

- Think about how historical data is handled in different SCD types.
- Consider which method allows for tracking changes over time within the same table.
